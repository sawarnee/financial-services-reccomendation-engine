About Recommendation System

Problem Statement

No system is available for suggesting a person on how to use their money for saving, where to invest and how to manage expenditures. The problem is to create a recommendation engine which inputs the data like last month bank statement and other financial and career details which will recommend the user to what they are overspending on and where they should stop spending (casinos, cinemas, excessive clothing, etc.). Further the recommendation engine will recommend where the user can reduce their spending (swapping groceries stores for discount stores)

Get Recommendations on the basis of:
  • Location
  • Age
  • Occupation
  • Gender
  • Monthly income
  • Monthly expenditure
